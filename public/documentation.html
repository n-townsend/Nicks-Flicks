Documentation for Nicks Flicks

In this application you will be able to sort through various movie genres,
view director information, and save any of your favorites to your user account.

Business Logic URL Endpoint HTTP Method Query Parameters Request Body Data Format Response Body Data Format
To allow the creation of individual user accounts /users POST None JSON of the new user in structure: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"} JSON of the new user in structure: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"}
Return a list of all movies /movies GET None None JSON of a defined movie
Return a movie by name /movies/:title GET None None JSON of a defined movie
Return data about a genre by name /movies/genres/:genre GET :genre None JSON of a specified genre
Return data about a director by name /movies/directors/:director GET :Name None JSON of a specified director
Return a list of all users /users GET None None JSON of all users
Return a single user by username /users/:username GET :username None JSON of specfic user
Allow users to change their information /users/:username PUT :id JSON of the new user in structure: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"} JSON of the new user in structure: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"}
Allow users to add movie to favorites list /users/:username/movies/:movieID POST :username, :movieID None JSON of the user
Allow users to remove movie from favorites list /users/:username/movies/:movieID DELETE :username, :movieID None Text confirmation that the movie has been removed
Allow a user to be removed /users/:username DELETE :username None Text confirmation that the user has been removed
